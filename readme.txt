A modern and user-friendly electric service booking system that helps customers schedule appointments with certified electric linemen quickly and efficiently.

== Description ==

Electric Lineman Booking is a professional booking plugin designed specifically for electric service providers. Easily allow your customers to book appointments with certified electric linemen through a simple, responsive booking form. Perfect for businesses in London and beyond looking for an efficient and reliable booking solution.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/electric-lineman-booking` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the settings page under the 'Electric Lineman Booking' menu to configure the plugin.
4. Place the booking form shortcode `[electric_lineman_booking_form]` on any page or post where you want the booking form to appear.

== Frequently Asked Questions ==

= Does this plugin support multiple linemen? =  
Yes, you can add and manage multiple electric linemen and their availability.

= Can I customize the booking form fields? =  
Yes, the form fields are customizable via the admin settings.

= Is the plugin compatible with the latest WordPress version? =  
Yes, it is tested up to WordPress 6.4.
